Jamaican Recipes App
Welcome to the Jamaican Recipes App! This web application allows users to explore a variety of delicious Jamaican recipes, share their own recipes, and interact with other users.

Features
Browse a collection of authentic Jamaican recipes, including main courses, appetizers, side dishes, and desserts.
Add new recipes to the collection, complete with name, category, and optional image.
View details of each recipe, including ingredients, cooking instructions, and user ratings.
Interact with other users by leaving comments and ratings on recipes.
User authentication and authorization system to ensure secure access to the application's features.
Responsive design for seamless user experience on desktop and mobile devices.
Technologies Used
Node.js
Express.js
MongoDB (or any other database system)
EJS (Embedded JavaScript) for templating
CSS for styling
bcrypt.js for password hashing
JSON Web Tokens (JWT) for user authentication
Other dependencies: body-parser, mongoose (for MongoDB), etc.
